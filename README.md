# WAAP Stats Datasource Plugin

Grafana datasource plugin for G-Core Labs WAAP statistics.

## Getting Started

### Prerequisites
- Node.js >= 22
- Docker and Docker Compose

### Development Setup

1. Install dependencies:
   npm install
Build the plugin in watch mode:

npm run dev
Start Grafana with the plugin (in another terminal):

npm run server
Access Grafana at http://localhost:3000
